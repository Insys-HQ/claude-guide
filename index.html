📋 Claude란?

Claude는 Anthropic이 만든 AI 비서입니다. 문서 작성, 데이터 분석, 번역, 코드 작성 등 다양한 업무를 도와줍니다.

데스크톱 앱 Claude 앱

컴퓨터에 직접 설치하는 프로그램입니다. Windows·Mac에서 사용하며, 내 컴퓨터 파일과 직접 연동하여 작업할 수 있습니다.

웹 버전 Claude 웹

크롬·엣지 등 브라우저에서 바로 사용합니다. 설치 없이 claude.ai에 접속하면 즉시 사용 가능합니다.

💻 데스크톱 앱 장단점

컴퓨터에 설치하여 사용하는 앱 버전의 특징입니다.

👍 장점

  • 내 컴퓨터 파일 직접 읽기·수정·생성
  • Cowork 모드로 문서 작업 자동화
  • Slack·Google 캘린더 등 외부 서비스 연동
  • 단축키로 어디서나 빠르게 호출
  • Word·PPT·Excel 파일 직접 생성·저장
  • 프로젝트별 맞춤 지시사항 설정

👎 단점

  • 별도 프로그램 설치 필요
  • Windows·Mac만 지원
  • 초기 파일 권한 설정이 다소 복잡
  • 컴퓨터 성능에 따라 속도 차이
  • 업데이트 시 재설치 필요한 경우
💡 앱 추천 대상: 문서 작업이 많은 분, 반복 업무 자동화를 원하는 분, 외부 도구와 연결하여 사용하고 싶은 분.

🌐 웹 버전 장단점

브라우저에서 바로 사용하는 웹 버전의 특징입니다.

👍 장점

  • 설치 없이 브라우저에서 즉시 사용
  • PC·태블릿·스마트폰 모든 기기 접속
  • 항상 최신 버전 자동 적용
  • Artifacts로 코드·차트 바로 미리보기
  • 가입 후 바로 사용, 초기 설정 간단
  • 스마트폰으로 대화 이어가기

👎 단점

  • 내 컴퓨터 파일 직접 접근 불가
  • Cowork 모드·플러그인 미지원
  • 브라우저 탭 전환으로 흐름 단절
  • 파일 직접 저장 기능 제한
💡 웹 추천 대상: 간단한 질문·답변 위주, 이동 중 사용, 설치 없이 빠르게 시작하고 싶은 분.
